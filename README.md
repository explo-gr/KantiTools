## Build Instructions
```bash
# Clone repo & install deps
git clone https://github.com/explo-gr/KantiTools.git
cd KantiTools
npm install

# Local Server (Expo Go)
npx expo start

# Run on emulator/simulator
npx expo run:android
npx expo run:ios

# Build the app
eas build -p android --local
eas build -p ios --local
```
